# Neumorphism Keyboard

Projeto de prática de front-end que consiste em um **teclado virtual interativo** com estilo visual **Neumorphism** (soft UI), desenvolvido com HTML, CSS.

## 📋 Sobre o projeto

Uma página que simula um teclado QWERTY clicável, onde cada tecla pressionada escreve o respectivo caractere em uma área de exibição (display) na parte superior da tela. O visual utiliza o estilo Neumorphism, caracterizado por sombras suaves que dão um efeito de relevo/extrusão nos elementos.

## 🖼️ Preview

A página exibe:

- **Display de texto:** área com a mensagem inicial "Clique nas teclas para escrever..."
- **Teclado virtual completo:**
  - Linha 1: Q W E R T Y U I O P
  - Linha 2: A S D F G H J K L
  - Linha 3: Z X C V B N M + tecla de Backspace (⌫)
  - Tecla de espaço (SPACE)
- **Estilo visual:** Neumorphism (sombras suaves, efeito 3D sutil, paleta clara)
